   # City Intelligence System

    An AI-powered agent that provides real-time weather and news information about cities using LangChain, Mistral AI, and Tavily search.

    ## Features

    - **Weather Information**: Get current weather conditions for any city in India
    - **News Updates**: Fetch the latest news about specified cities
    - **Human-in-the-Loop**: Confirm tool usage before execution
    - **Multi-turn Conversation**: Maintain context across multiple queries

    ## Requirements

    - Python 3.8+
    - `langchain-mistralai`
    - `langchain-core`
    - `tavily-python`
    - `requests`
    - `python-dotenv`
    - `rich`
